# protoc-gen-go-gin-errors

protoc-gen-go-gin-errors is a protoc plugin that generates errors for gin.

## Install

```shell
$ go install github.com/trumanwong/protoc-gen-go-gin-errors@latest
```

## Usage

api/app/v1/app_error.proto

```proto    
syntax = "proto3";

package api.app.v1;

option go_package = "gin-layout/api/app/v1;v1";
option java_multiple_files = true;
option java_package = "api.app.v1";

import "errors/errors.proto";

enum AppServiceErrorReason {
	option (errors.default_code) = 400;

	UNKNOWN_ERROR = 0;
	LOGIN_FAILED = 1 [(errors.code) = 401];
	FORBIDDEN = 2 [(errors.code) = 403];
	NotFound = 3 [(errors.code) = 404];
	NotAcceptable = 4 [(errors.code) = 406];
	Conflict= 5 [(errors.code) = 409];
	Internal = 6 [(errors.code) = 500];
}
``` 

```shell
$ protoc.exe --proto_path=. --proto_path=./third_party --go_out=paths=source_relative:. \
    --go-grpc_out=paths=source_relative:. \
    api/app/v1/app_error.proto
```
